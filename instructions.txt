new folder "site"
move the data.json and index.php into site folder
new Dockerfile file
write in Dockerfile:
    FROM (Recupera l'immagine di apache)
    COPY (Copia i tuoi file della cartella nella "htdocs" del container apache)
    RUN (Fornisci tutti i permessi possibili per il file data.json)
    EXPOSE (Esponi il docker su una porta)

## Build the Docker container ##
docker build -t site site/

## Run and Stop the Docker
docker run -d site
docker stop exciting_mccarthy

## Run and Stop the Docker as a Service
docker run -p 8080:80 site
docker stop exciting_mccarthy

## Run and Stop the Docker as a Service and Demon
docker run -dp 8080:80 site
docker stop exciting_mccarthy

## Remove and Rebuild the Docker container ##
docker rm exciting_mccarthy
docker build -t site site/
